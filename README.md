# 特性
- 可选将本机 DNS 请求转发到本地 AdGuardHome 服务器
- 使用 [秋风广告规则](https://github.com/TG-Twilight/AWAvenue-Ads-Rule) 过滤广告，轻量，省电，少误杀
- 可从 `http://127.0.0.1:3000` 访问 AdGuardHome 控制面板，支持查询统计，修改 DNS 上游服务器以及自定义规则等功能
